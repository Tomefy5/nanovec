# Security Policy

## Supported Versions

We currently support the following versions of NanoVec with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please let us know immediately. 

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to [INSERT EMAIL ADDRESS].

We will acknowledge receipt of your vulnerability report as soon as possible and strive to provide a regular status update until the vulnerability is addressed.
